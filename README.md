# Image Converter GUI

## 專案簡介

**Image Converter GUI** 是一個使用 Python 製作的圖形化工具，提供圖片批量轉換與尺寸修改功能，並支援即時圖片預覽。  

特點：

- 支援 **PNG、JPG、JPEG、WEBP** 格式互轉  
- 批量選擇檔案或資料夾（包含子資料夾）  
- 支援 **修改圖片尺寸**（保持寬高比例）  
- 預覽圖片及詳細屬性（檔案路徑、尺寸、大小、副檔名等）  
- 簡單易用的 GUI，適合沒有 Python 基礎的使用者  

---

## 功能介紹

### 1️⃣ 檔案選擇

- 單一檔案選擇  
- 多檔案選擇  
- 資料夾選擇（可包含子資料夾）  
- 移除選取檔案  

### 2️⃣ 格式轉換

- 選擇輸出格式：PNG / JPG / JPEG / WEBP  
- 支援批量轉換  
- 可指定輸出資料夾  

### 3️⃣ 尺寸修改

- 可以指定新寬或新高，自動保持寬高比例  
- 支援單檔或批量修改  

### 4️⃣ 圖片預覽

- 左側顯示縮圖  
- 右側顯示圖片屬性（檔案路徑、尺寸、大小、副檔名、索引）  
- 上一張 / 下一張按鈕可快速切換圖片  
- 長路徑自動換行顯示
